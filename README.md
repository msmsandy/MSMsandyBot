# MSMsandyBot 

They said I was a bot, so here I am.

## Setup 

### Install 

```
brew install npm 
npm install 
```

### .env 

There should be a `.env` in the root folder. 

```
DEBUG=true

BOT_TOKEN=

MONGODB_DATABASE=
MONGODB_PASSWORD=
MONGODB_SERVER=
MONGODB_USER=
```

### Run 

```
./run 
```
